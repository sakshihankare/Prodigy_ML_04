🔹 About the Project

This project focuses on developing a Hand Gesture Recognition system using Convolutional Neural Networks (CNN). The model is trained on image data to classify different hand gestures, enabling gesture-based interaction systems.

🔹 Description

The system processes image data, performs preprocessing, and trains a deep learning model to recognize and classify hand gestures. A lightweight CNN architecture was used to ensure faster training while maintaining high performance. The model achieved very high accuracy on the test dataset, demonstrating strong learning capability within controlled conditions.

🔹 Key Features
Image-based gesture classification
Efficient preprocessing and normalization
Lightweight CNN model for faster execution
Performance evaluation using accuracy and confusion matrix

🔹 Tech Stack
Python
TensorFlow / Keras
OpenCV
NumPy, Matplotlib, Seaborn
Scikit-learn

🔹 Results

The model achieved high accuracy on the test dataset. Due to the structured nature of the dataset (consistent background and lighting), the model performs strongly on similar data distributions.

Human-Computer Interaction
Gesture-based control systems
Touchless interfaces
Assistive technologies
